## Visual Stidio中Gitpush请求失败
报错：unable to access 'https://github.com/MyJubilation/TecLearningBLogs.git/': Failed to connect to github.com port 443 after 21138 ms: Couldn't connect to server
Sol：通过在终端执行以下命令，可以取消 Git 的代理设置：    
    git config --global --unset http.proxy     
    git config --global --unset https.proxy
